# DGS Hud 
**A Simple Hud Based DGS ([Thisdp's DirectX Graphical User Interface System](https://github.com/thisdp/dgs)) For MTA Servers**

# Installation 
* Just copy  DGS/DGS-Hud into you resources directory
* start dgs
* start dgs-Hud
* Have Fun
